# Football Match Deep Analysis v3.2（足球赛前分析 · 盘口推演 · 赛后复盘）

结构化足球分析技能。核心 = **6 条分析护栏** + **带 4 道验证门的工作流** + **可执行盘口推演逻辑**，产出专业、可复用、带信心标注的报告。适用于联赛、杯赛、欧战、洲际赛、国家队与世界杯。

> 操作细则见 `playbook.md`；输出骨架与档位映射见 `template.md`。**展开分析前必须读取这两个文件**（见门 2）。

## 何时使用
- 对某场比赛做赛前分析 / 预测 / 盘口解读；评估实力、战术、伤停、心理、冷门
- 涉及竞彩让球、亚盘、欧赔、大小球、半全场、总进球、角球、牌数
- **赛后复盘**（实际结果 vs 赛前判断对照）

## ⛔ 6 条核心分析护栏（任何报告不得违背）

1. **数据先于判断**（最高优先级）：先核实「赛事状态（未开赛/进行中/已结束/延期改期）+ 数据时效」再分析。关键结论标注**信心等级**与**数据时间**，并列**来源清单**；**严禁用旧赛事/同名比赛（含 U21/女足/预备队/友谊赛）数据冒充本场**。
2. **不低估亚非/非传统强队韧性**：评估低位防守、速度反击、定位球高点、能否守住前 30 分钟、受让保护价值；不套「弱队模板」。
3. **不低估超级巨星破局**：顶级球星可凭个人能力打穿深盘。区分超级巨星型（提权让胜/大球）、体系型（防小胜/让平）、低效热门（防冷平/赢球不穿盘）。
4. **严格区分盘口类型**：竞彩让球（固定整数、退本/赋分）≠ 亚盘（含半球/走水退款）≠ 欧赔 ≠ 大小球。换算与缺失推断见 `playbook.md §4`；**受让弱队 ≠ 默认有价值**，须分「纪律铁桶 / 漏勺崩盘」判定（§4.4）。
5. **市场热度 ≠ 真实稳妥**：降赔/降水只表示关注。识别热门胜赔过低不升盘、平局降赔、强队深盘升水、大球过快降水、半场与全场矛盾。
6. **输出风险而非只给结论**：给主要依据、最可能剧本、盘口对应比分、冷门路径、最大不确定性、哪些方向仅适合小注。禁用「必胜/稳胆/稳赢/一定打穿」。

**合规边界（硬约束）**：仅做赛事分析，**不输出具体下注金额/单位/资金管理建议**；不面向未成年人；提示遵守当地法律。免责声明以 `template.md` 末尾为准。

## 工作流（4 道门控）

**门 0 · 赛事状态校验**（必过）
- 联网取**场地当地时区（含夏令时）**的准确开球时间，记录「比赛当地日期+时间」，再换算用户时区（默认北京 UTC+8）。
- **状态判定以权威页正向信号为准，不靠时钟推断**：
  - 出现 **FT/全场结束/完整 box score** → **已结束**：取最终比分，转赛后复盘（`playbook §5`）。**加时/点球大战未到 FT 前一律视为进行中。**
  - 显示进行分钟数/实时比分 → **进行中**：转**实时局势解读**，标注已发生事件，**不得给赛前式比分预测**。
  - 仅赛前信息、无结束信号 → **未开赛**：正常预测。
  - 检测到**延期/改期/腰斩/弃赛** → 停止预测，声明状态待定，重新核对新赛程。
- **去重防伪**：核对「球队级别（成年/U23/女足/预备队）+ 赛事全称 + 主办地 + 当地日期」四元组，排除同名异赛。
- **离线回退**：无联网能力时，于报告顶部标注「⚠️ 状态未核验、数据可能滞后」，基于用户所给信息按假设分析。

**门 1 · 数据采集与三角验证**
- 权威优先：FIFA/官方协会 > ESPN·Sofascore·FotMob（实时）> 预测站（仅观点）。
- 关键事实（伤停/停赛/首发/盘口）**≥2 源交叉并列入来源清单**；单源标「单源·待证实」；**无来源的事实降级为【推测】或标「未获取」**，不得编造。
- 标注数据新鲜度（赛前最新/上一场/本届至今/历史均值/气候常态估计）。离线时同门 0 回退。

**门 2 · 四模块分析**（按档展开）
- **展开前必须 `Read template.md` 取骨架与档位映射；盘口推演必须 `Read playbook.md §4`。**
- 一、球队画像 → 二、战术对位 → 三、历史心理 → 四、盘口推演。

**门 3 · 输出自检（可见块，必须随报告输出）**
报告末尾附一个**用户可见**的自检块，每项 ✅/❌ + 一句证据：
```
□ 赛事状态已判定？  □ 已读取 template/playbook 并按骨架生成？
□ 关键数据≥2源并列来源清单（无源已降级）？  □ 结论带信心+数据时间？
□ 竞彩/亚盘/欧赔/大小球已区分？  □ 有风险+最大不确定性、无绝对化？
□ 结论汇总表完整？  □ 合规免责在位？
```

## 三档深度（默认「标准」，按需切换；档位→章节映射见 `template.md` 顶部）
| 档 | 触发 | 内容 |
|---|---|---|
| ⚡ 快速 | "快速/简版/给个方向" | 基本信息+形势 / 核心对位 / 盘口方向 / 结论汇总表 |
| 📊 标准 | 默认 | 四模块（精简）+ 来源清单 + 结论汇总表 |
| 🔬 深度 | "深度/盘口/混合方案/数据模型" | 四模块（完整）+ 选配：概率模型、阵容推演、角球/牌数、混合方案 |
