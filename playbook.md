# Playbook — 赛事校验 · 数据采集 · 盘口推演 · 赛后复盘

SKILL.md 工作流的可执行细则。生成报告时按需查阅。

---

## 1. 门 0 · 赛事状态校验（防止"预测"已决比赛 / 误用旧数据）

**1.1 取准确开球时间**
- 搜索权威开球时间，**先取场地当地时区（含夏令时 CEST/BST/EDT 等）**，标注源时间是 UTC 还是 local。
- 记录「**比赛当地日期 + 时间**」，再换算用户时区（默认北京 UTC+8）。跨日深夜场注意日期 ±1。

**1.2 状态判定——以权威页正向信号为准，不靠时钟推断**
| 观测到 | 状态 | 动作 |
|---|---|---|
| FT / 全场结束 / 完整 box score | **已结束** | 取最终比分 → 赛后复盘（§5） |
| 实时比分 / 进行分钟数 | **进行中** | 转实时局势解读，标已发生事件，**不给赛前式比分预测** |
| 仅赛前信息、无结束信号 | **未开赛** | 正常预测 |
| 延期/改期/腰斩/弃赛(walkover) | **待定** | 停止预测，声明状态待定，重新核对新赛程 |
> ⚠️ **加时/点球大战在出现 FT 前一律算"进行中"**，不可因"开球已超 2.5h"就去抓"最终比分"——淘汰赛加时+点球+VAR 补时常达 3h+，按时钟推断会抓到 90 分钟临时比分当终局。

**1.3 去重防伪（核对四元组）**
任何"比分/结果"必须核对：**① 球队级别（成年/U23/U21/女足W/预备队）② 赛事全称 ③ 主办地/国 ④ 当地日期**。
- 警惕同名异赛：同一对阵的友谊赛/往季/不同组别（多支同名俱乐部、男足≠女足）。
- 判据用**正向结束信号**（FT/box score），不用"页面有无赔率"（赔率页可能缓存过期，或临开赛封盘下架→无赔率却未开赛，均会误判）。

**1.4 离线回退**：无联网能力时，报告顶部标「⚠️ 状态未核验、数据可能滞后」，基于用户信息按假设分析，不杜撰实时数据。

## 2. 门 1 · 数据源分级与三角验证
| 级别 | 来源 | 用途 |
|---|---|---|
| **A 权威** | FIFA 赛事中心、官方协会/俱乐部、UEFA | 赛程、最终结果、官方首发、停赛 |
| **B 实时聚合** | ESPN、Sofascore、FotMob、Flashscore | 实时比分、预计首发、伤停、统计、盘口线 |
| **C 观点** | 预测站、博彩预览、媒体 preview | 倾向参考，**不可单独作事实依据** |

- 关键事实 **≥2 源**，冲突按 A>B>C 取信并标分歧；单源标「单源·待证实」。
- 预计首发列 2-3 家并标「预测，以官宣为准」。

---

## 3. 信心标注 + 来源清单（强制留痕，防"贴标签不核对"）
- 标签：**【已证实】**(官方/多源一致) · **【可能】**(有据推断) · **【推测】**(弱证据/假设)。
- 时间锚：`(赛前最新)`/`(上一场)`/`(本届至今)`/`(历史均值)`/`(气候常态估计)`。
- **报告必含「来源清单」小节**：每条关键事实（伤停/停赛/首发/盘口）后附 **≥2 个来源名 + 抓取日期**。
- **无来源支撑的事实，一律降级【推测】或标「未获取」，不得以【已证实/可能】蒙混。**

## 4. 盘口换算与推断

### 4.1 亚盘未公布时——由欧赔 1X2 反推（**先去抽水→剥离平局→用主胜净概率查表**）
> ⚠️ 不能只看主胜单赔！三路 1X2 中平局吃掉了部分概率，单看主胜赔会系统性高估热门、把让球推太深。

**步骤**
1. 三路赔率各转隐含概率（美式：负数 `|O|/(|O|+100)`；正数 `100/(O+100)`；小数 `1/O`）。
2. 三者相加 >100%（抽水），按比例归一 → 净概率 **P(主)、P(平)、P(客)**。
3. 用 **P(主胜) 净概率** 查表定让球主线：

| P(主胜) 净概率 | 亚盘 主队让球 |
|---|---|
| ~38–44% | 平手 0 |
| 44–50% | -0.25 |
| 50–56% | -0.5 |
| 56–62% | -0.75 |
| 62–68% | -1.0 |
| 68–74% | -1.25 |
| >74% | -1.5+ |

> **双案例校验**：德国(主-180/平+365/客+540)→去抽水 P(主)≈63%→让 **-1.0**（实盘 -1/-1.25 ✅）；墨西哥(主-106/平+225/客+300)→P(主)≈48%→让 **-0.25**（实盘 平手/-0.25 ✅）。
> **回退护栏（量化）**：若 **P(平)≥30%** 或 **P(主)−P(客)<22 个百分点**（均势/低进球对决）→ 查表结果**回退一档**（-0.5→-0.25、-0.25→平手），因高平局概率压缩净胜球期望。校验：+110/+230/+240→P(主)44%、主客差仅 17pp→回退为**平手**（而非 -0.25）。
> **镜像（客队为热门）**：若 P(主)<38%（表外）→ 改用 **P(客)** 净概率查同表，符号取主队受让（如 P(客)64%→主队 +1.0）。
> **其它修正**：中立场/客场 → 主队让球减半档；两回合赛按晋级需求另判（见 §6）。

### 4.2 竞彩让球 与 亚盘 区别（拆开看，不可混 — 护栏#4）
**竞彩「主队让 1」**（固定整数、按规则退本/赋分）：
| 实际比分 | 竞彩结果 |
|---|---|
| 主胜 ≥2 球 | 让胜 |
| 主胜 1 球 | 让平（让 1 走盘） |
| 平 / 客胜 | 让负 |

**亚盘结算（对照，规则不同）**：
| 盘口 | 主胜 1 球 | 主胜 ≥2 球 | 平 |
|---|---|---|---|
| 主 -1.0 | **走水退款** | 赢 | 输 |
| 主 -1.5 | **输半** | 赢 | 输 |
> 即：竞彩"让 1 走盘"≠ 亚盘"-1.5 输半"。同一"主胜 1 球"在两套玩法结算完全不同，报告中务必分开表述。

### 4.3 市场信号速查
| 信号 | 含义 | 风险方向 |
|---|---|---|
| 热门胜赔降、亚盘不升 | 钱进热门但庄不抬盘 | 防赢球不穿盘 |
| 平局赔率下降 | 嗅到平局/小胜 | 防冷平 |
| 强队深盘水位升高 | 上盘承压 | 受让有保护 |
| 大球快速降水 | 大球过热 | 反向防小球 |
| 受让方持续降水 | 资金挺弱队 | 冷门预警 |
| 半场盘 vs 全场盘矛盾 | 节奏预期不一 | 半全场慎选/只小注 |

## 5. 赛后复盘模式（门 0 判为「已结束」时启用，不再给"预测"）
1. **最终结果**：比分 + 进球过程（射手/分钟）+ 关键事件（红牌/点球/伤退/加时点球）。
2. **赛前判断对照**：逐项 ✅命中 / ❌落空（胜负、比分、大小球、盘口方向）。
3. **偏差归因**：伤停/阵容变化、战术误判、运气（中框/越位/裁判）、盘口误读。
4. **经验沉淀**：可迁移规律，供下次分析。

## 6. 输入契约（门 0 收集）
| 项 | 必需 | 缺失处理 |
|---|---|---|
| 对阵（主/客或中立） | 是 | 向用户确认 |
| 赛事 / 轮次 | 是 | 联网推断+确认 |
| 开球时间 | 是 | 联网取权威（§1.1） |
| 地点/主客中立 | 是 | 联网取 |
| 盘口信息 | 否 | 联网取；仍无则按 §4 反推并标注 |
| **两回合赛**：首回合比分/晋级所需净胜球 | 视赛制 | 淘汰赛次回合必取——决定战意与大小球预期 |
| 其他背景（战意/德比） | 否 | 联网补，标「未获取」 |
