# ⚽ Football Match Deep Analysis Skillv2

> 一个用于足球比赛赛前深度分析的结构化 Prompt 模板，适用于联赛、杯赛、欧战、洲际赛事、国家队赛事和世界杯等高强度比赛场景。
> 目标是帮助分析师或 AI 模型从球队基本面、战术结构、关键对位、伤停体能、心理因素、盘口走势和风险控制等维度，输出一份专业、清晰、可复用的赛前分析报告。

---

## 📌 Skill 简介

本 Skill 适用于足球比赛的赛前深度分析，重点覆盖：

* 球队整体实力、排名形势与近期状态
* 战术体系、攻防结构与关键对位
* 伤停、轮换、赛程、体能和旅行影响
* 历史交锋、心理因素与比赛动力
* 比分预测、胜负倾向、进球数判断
* 盘口走势、赔率变化、让球逻辑与风险提示
* 可选扩展：竞彩让球、亚盘、大小球、半全场、总进球数、角球、牌数和冷门风险

本 Skill 特别强调：
**强队不等于稳胜，弱队不等于无抵抗；盘口方向必须结合比赛风格、球星破局能力、受让方韧性和市场热度综合判断。**

---

## ⚠️ 核心分析原则

### 1. 不低估亚非球队韧性

在世界杯、洲际杯和国家队大赛中，亚洲、非洲球队往往具备更强的防守纪律、身体对抗、拼抢强度和心理韧性。面对传统强队时，不应简单套用“弱队模板”。

需要重点评估：

* 是否具备低位防守执行力
* 是否有速度型反击点
* 是否有定位球高点
* 是否能守住前 30 分钟
* 是否具备被压制后仍不崩盘的能力
* 受让盘口是否存在保护价值

### 2. 也不能低估超级巨星破局能力

面对摆大巴球队时，拥有超级巨星的强队仍可能打穿盘口。姆巴佩、梅西、哈兰德、凯恩、贝林厄姆、维尼修斯等球员，可能通过个人能力、定位球、禁区终结或最后一传直接打破密集防守。

分析强队深盘时，需要区分三类球队：

| 类型      | 特征             | 分析倾向            |
| ------- | -------------- | --------------- |
| 超级巨星型强队 | 拥有稳定破局点和终结点    | 可提高让胜、大球、3球以上权重 |
| 体系型强队   | 控球或压迫强，但缺少绝对爆点 | 防小胜、让平、小3.5     |
| 低效热门队   | 纸面强但终结不稳或状态一般  | 防冷平、防赢球不穿盘      |

### 3. 竞彩让球与亚盘必须区分

分析让球时必须说明盘口类型：

| 类型   | 示例                | 判断方式                    |
| ---- | ----------------- | ----------------------- |
| 竞彩让球 | 法国让1球、阿根廷让1球      | 强队赢2球以上为让胜，赢1球为让平，不胜为让负 |
| 亚洲盘口 | 法国-1.25、塞内加尔+1.5  | 需结合水位、升降盘、赢半输半逻辑判断      |
| 欧洲赔率 | 主胜、平局、客胜          | 判断市场胜负倾向与冷门防范           |
| 大小球  | 大2.5、小2.5、大3、小3.5 | 判断进球预期与节奏变化             |

禁止将“竞彩让1”和“亚盘-1.5”混为一谈。

### 4. 市场热度不等于真实稳妥

赔率下降、热门降水、大球降水，说明市场关注该方向，但不代表该方向一定更稳。

需要重点识别：

* 热门胜赔率过低但让球不升盘
* 平局赔率下降
* 强队深盘水位升高
* 大球降水过快导致过热
* 受让方水位持续下降
* 半场盘口与全场盘口出现矛盾
* 盘口支持强队胜，但不支持强队大胜

### 5. 输出必须包含风险，而不是只给结论

任何赛前分析都必须说明：

* 主要判断依据
* 最可能剧本
* 盘口对应比分
* 潜在冷门路径
* 最大不确定性
* 哪些方向只适合小注或进取方案

避免使用“必胜”“稳胆”“一定打穿”“稳赢”等绝对化表达。

## 🚀 使用方法

将下方模板中的 `【】` 内容替换为实际比赛信息，然后直接发送给分析师或 AI 模型即可。

建议每一场比赛单独使用一次模板，以便获得结构统一、逻辑清晰的赛前报告。
